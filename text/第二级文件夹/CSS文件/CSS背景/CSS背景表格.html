背景总结
属性 作用 值
background-color 背景颜色 预定义的颜色/十六进制/RGB代码
background-images 背景颜色 url（图片路径）
background-repeat 是否平铺 repeat/no-repeat/repeat-x/repeat-y
background-position 背景位置 length/position 分别是x和y坐标，切记 如果有精确数值单位，则必须按照先x后y的写法
background-attachment 屏幕固定还是滚动 scrll/fixed
背景简写 更简单 背景颜色 背景图片地址 背景平铺 背景滚动 背景位置;他们没有顺序
背景透明 让盒子半透明 background:rgba(0,0,0,0.3); 后面必须是4个值
